Bird Atlas Generator (BAG)

Here is the first publicly available comprehensive Bird Atlas Generator (BAG) based on the North American Breeding Bird Survey data. It has been developed at in-situ mediation for conservation practitioners and the bird-aware. BAG assists bird research communities with powerful tools for generating spatially and temporally explicit visualizations of bird population trends.

As input, SaveBirds receives the year interval and a map in shapefile (.shp) format. Then SaveBirds provides the user with a zip file that contains hundreds of CSV files, each related to one species, where each CSV contains the modeled species trend in the year interval across specified shapefile geometry defined areas.

For each species, the BAG creates a Minimum Convex Polygon (MCP) for the routes in which the selected bird species has been seen in the year interval across North America. Then the BAG overlaps the MCP on a raster. By default, we make the cell size of the raster 25 miles x 25 miles.
